Introduction

JsonTransform is a library helping with transformation of JSON objects. You use JavaScript-like expressions to instruct JSON data structure selection, filtering and reduction. Here we can see a simple example which is purely JS compliant and very basic, not showing the true power of JsonTransform.

As you can see you have a field Expression where you provide your transformation expression, and dataset where you provide JSON object against which expression is executed. Finally, after clicking Transform button, you will get result in the DatasetTransformed. More structure result you can see in the field DatasetTransformedHTML.

In the background, something similar to 

$("#DatasetTransformed").val(JSONTransform( $("#dataset").val(), $("#expression".val() ) );
is happening.

You can try to change expression and see result change, and then you can navigate to the next example
